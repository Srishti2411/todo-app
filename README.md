# To-Do App

A simple and interactive To-Do web application built using HTML, CSS, and JavaScript.

## 🔗 Live Demo
https://srishti2411.github.io/todo-app/

## 🚀 Features
- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as complete
- Data saved using localStorage

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📌 Description
This project demonstrates DOM manipulation, event handling, and local storage in JavaScript.
